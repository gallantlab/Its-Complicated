# Experiment-Building concepts

This section is structured into three parts. First, it will provide an overview of the paradigm difference in experimental design, contrasting it with traditional neuroscience experiments. Second, it will provide an overview of game engine concepts that are relevant for building experiments. Finally, it will describe some specific tools that we have developed for these experiments in the hopes that they will be reusable for you. We write this article presuming that the user has some topic or scientific question in mind to study, and are now looking at the implementation.
